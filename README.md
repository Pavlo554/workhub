# WorkHub

**WorkHub** — десктопний CRM-додаток для фрілансерів, бухгалтерів, SMM-спеціалістів та салонів краси. Все що потрібно для роботи — в одному місці, без підписок на сторонні сервіси.

> Розробник: AtomicX Digital ⚡️ · Версія 1.0 · Активна розробка

---

## Для кого

| Ніша | Профіль |
|------|---------|
| 💻 Фрілансер | Проекти, рахунки, контракти, таймер, портфоліо |
| 📊 Бухгалтер / ФОП | Рахунки, податки, аналітика, курси валют |
| 📱 SMM / Маркетолог | Контент-план, акаунти, аналітика клієнтів |
| 💅 Салон краси | Записи, послуги, склад, персонал, фінанси |

---

## Що реалізовано ✅

### Ядро системи
- **Авторизація** — реєстрація / вхід через Firebase Auth (email + пароль)
- **Онбординг** — вибір ролі (власник / учасник команди) та ніші при першому вході
- **Дашборд** — персоналізований під нішу: статистика, останні клієнти, рахунки, задачі, швидкий перехід до модулів
- **Профіль** — ім'я, email, Telegram, аватар
- **Налаштування** — теми (темна / світла / системна), акцентний колір (5 варіантів), нотифікації, вигляд

### Клієнти та продажі
- **Клієнти** — база клієнтів: додавання, редагування, пошук, статус (active / inactive), теги
- **Проекти** — прив'язка до клієнта, статус, дедлайн, бюджет
- **Рахунки (Invoices)** — виставлення, статуси (draft / sent / paid / overdue), перегляд
- **Контракти** — шаблони та зберігання договорів
- **Портфоліо** — роботи фрілансера з описом та посиланнями

### Задачі та управління
- **Задачі** — створення, пріоритет, дедлайн, статус, прив'язка до клієнта/проекту
- **Канбан** — дошка зі стовпцями (До виконання / У процесі / Готово), drag & drop
- **Таймер** — відстеження робочого часу, фіксація активних додатків, перерви з типами (обід / кава / покурити тощо), зміна задачі під час сесії, таймлайн сесії, збереження в хмару

### Фінанси
- **Фінанси** — доходи, витрати, баланс
- **Податковий календар** — ключові дати для ФОП / бухгалтера
- **Курси валют** — актуальні курси
- **Аналітика клієнтів** — статистика по клієнтах і доходах

### Контент і комунікації
- **Контент-план** — планування публікацій для SMM
- **Акаунти** — зберігання облікових записів соцмереж

### Документи та безпека
- **Документи** — зберігання файлів і документів
- **Шаблони** — шаблони документів і листів
- **Паролі** — зашифрований менеджер паролів
- **Нотатки** — особисті нотатки

### Для салону краси
- **Записи (Appointments)** — розклад клієнтів
- **Послуги** — каталог послуг з цінами
- **Склад (Warehouse)** — залишки товарів та матеріалів
- **HR** — управління персоналом

### Команда
- **Workspace** — робочий простір для команди
- **Запрошення** — додавання учасників через invite-посилання
- **Ролі** — власник / менеджер / учасник

### Підтримка та адміністрування
- **Підтримка** — тікети: баг-репорти, пропозиції, звернення; відповідь адміна; статус заявки
- **Новини** — адмін публікує оголошення, юзери бачать у вкладці "Новини"
- **Адмін-панель** — управління юзерами, плани, бани, видача прав адміна, тікети підтримки, новини
- **API Ключі** — зберігання API-ключів для зовнішніх сервісів
- **Підписка** — сторінка планів (free / pro / business)
