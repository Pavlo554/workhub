================================================================
  WORKHUB — CRM для малого бізнесу
  Повний огляд проекту
================================================================

ТЕХНІЧНИЙ СТЕК
──────────────
- Electron (v28) — десктопний застосунок (Windows / Mac / Linux)
- Vanilla JS (ES Modules) — без фреймворків, чистий JS
- Firebase Firestore — база даних в реальному часі
- Firebase Auth — автентифікація користувачів
- jsPDF — генерація PDF-рахунків та договорів
- electron-store — локальне зберігання налаштувань
- crypto-js — шифрування паролів
- Chart.js — графіки в звітах та кабінеті
- Stripe / LiqPay — платіжні системи (підписка)

================================================================
  МОДУЛІ (src/renderer/modules/)
================================================================

01. КЛІЄНТИ (clients/)
    - Список клієнтів з пошуком та фільтрами
    - Картка клієнта: контакти, статус, нотатки, теги
    - Форма створення / редагування клієнта
    - Фільтрація: активні / неактивні / архів
    - Зв'язок з рахунками, проектами, задачами

02. ПРОЕКТИ (projects/)
    - Канбан + список проектів
    - Картка проекту: назва, клієнт, статус, бюджет, прогрес
    - Трекер часу (time-tracker.js) — запис годин по проекту
    - Статуси: Активний / Призупинений / Завершено / Скасовано
    - Прогрес-бар виконання

03. РАХУНКИ (invoices/)
    - Список рахунків з фільтрами (оплачено / очікується / прострочено)
    - Створення рахунку: клієнт, сума, валюта, спосіб оплати
    - Способи оплати: Картка / Наличка / Крипта
    - Генерація PDF-рахунку (invoice-pdf.js + jsPDF)
    - Конструктор рахунку (invoice-builder.js)
    - Статистика: Дохід / Неоплачені / Прибуток / Очікуються
    - Summary-картки з іконками та кольоровим акцентом

04. ДОГОВОРИ (contracts/)
    - Список договорів
    - Шаблони договорів (templates.js)
    - Генерація PDF-договору (contract-pdf.js)
    - Статуси: Чернетка / Активний / Підписаний / Завершено

05. ЗАДАЧІ (tasks/)
    - Список задач з пріоритетами та статусами
    - Картка задачі: назва, клієнт, дедлайн, пріоритет
    - Статуси: Нова / В роботі / Виконана
    - Пріоритети: Низький / Середній / Високий / Критичний

06. ФІНАНСИ (finances/)
    - Облік доходів та витрат
    - Категорії транзакцій
    - Звіти по фінансах (reports.js)
    - Інтеграція з податковим календарем (tax-calendar.js)

07. ПОДАТКОВИЙ КАЛЕНДАР (tax-calendar/)
    - Календар сплати податків (ФОП, ТОВ)
    - Відмітка сплачених / несплачених
    - Нагадування про дедлайни
    - Фільтри по типу: ЄСВ / ЄП / ПДВ та ін.
    - Бічна панель: найближчий дедлайн, статистика, поради

08. РОЗКЛАД / ЗАПИСИ (appointments/)
    - Календарний вигляд (тиждень / список)
    - Запис клієнта: дата, час, послуга, ціна, контакт
    - Статуси: Заплановано / Виконано / Скасовано
    - Зв'язок із модулем Послуг

09. ПОСЛУГИ (services/)
    - Каталог послуг з категоріями та цінами
    - Категорії управляються користувачем (Firebase)
    - Автосід дефолтних категорій при першому запуску
    - Картка послуги: назва, категорія, ціна, тривалість, опис
    - Два режими: "Послуги" та "Категорії" (вкладки)
    - Іконки та кольори для кожної категорії

10. КОНТЕНТ-ПЛАН (content-plan/)
    - Планування постів для соцмереж
    - Прив'язка до акаунтів (accounts.js)
    - Картка поста (post-card.js)
    - Статуси: Чернетка / Запланований / Опублікований

11. АКАУНТИ (accounts/)
    - Зберігання соцмереж та онлайн-акаунтів бізнесу
    - Платформи: Instagram, Facebook, TikTok, YouTube та ін.
    - Швидкий доступ до посилань

12. ПАРОЛІ (passwords/)
    - Зашифрований менеджер паролів (crypto-js AES)
    - Категорії паролів
    - Копіювання в буфер з маскуванням

13. НОТАТКИ (notes/)
    - Простий редактор нотаток
    - Пошук по нотатках
    - Прив'язка до клієнта / проекту

14. ДОКУМЕНТИ (documents/)
    - Зберігання та управління файлами
    - Типи: PDF / Word / Excel / Зображення / Відео / Аудіо та ін.
    - Завантаження, перегляд, видалення
    - Фільтрація по типах

15. API & ІНТЕГРАЦІЇ (api-keys/)
    - Зберігання API-ключів та токенів
    - Інтеграції з зовнішніми сервісами

16. ТАЙМЕР (timer/)
    - Трекер робочого часу
    - Старт / пауза / зупинка
    - Зв'язок з проектами та задачами

17. KANBAN (kanban/)
    - Kanban-дошка для управління задачами
    - Колонки: Нові / В роботі / Перевірка / Виконано
    - Drag & Drop карток

18. ШАБЛОНИ (templates/)
    - Бібліотека шаблонів документів
    - Швидке створення нових документів на базі шаблону

19. СКЛАД (warehouse/)
    - Облік товарів / матеріалів / інструментів
    - Категорії: Матеріали / Інструменти / Товари / Обладнання
    - Кількість, ціна, залишки

20. ПОРТФОЛІО (portfolio/)
    - Каталог виконаних робіт
    - Типи: Веб / Дизайн / SMM / Відео / Фото та ін.
    - Зв'язок з клієнтом

21. ПЕРСОНАЛ / HR (hr/)
    - Список співробітників
    - Контакти, посада, дата прийому
    - Управління командою

22. АНАЛІТИКА КЛІЄНТІВ (client-analytics/)
    - KPI по клієнтах
    - Джерела залучення клієнтів
    - Рейтинги та тренди

23. ВАЛЮТИ (currency/)
    - Курси НБУ в реальному часі
    - Крос-курси валют
    - USD / EUR / інші

24. ЗВІТИ (reports/)
    - Split-panel: вертикальна навігація + контент
    - 4 вкладки:
      • Метрики — KPI, графік доходу за 6 міс., статус задач (Chart.js)
      • ТОП клієнти — рейтинг по виплатах
      • Задачі — аналіз по виконаних задачах + фільтр по періоду
      • Порівняння місяців — таблиця + графік за 6 міс.

25. ПІДТРИМКА (support/)
    - Split-panel: список заявок ліворуч, деталі праворуч
    - Типи: Bug Report / Пропозиція / Підтримка
    - Пріоритети: Низький / Середній / Високий / Критичний
    - Статуси: Нова / Відкрита / В роботі / Вирішено / Закрита
    - Треди відповідей (user ↔ адмін)
    - Вкладка "Новини" — анонси від WorkHub

================================================================
  СИСТЕМА ПЛАНІВ / ПІДПИСКИ
================================================================

  FREE     — базовий функціонал, обмежена кількість записів
  PRO      — повний функціонал, пріоритетна підтримка
  BUSINESS — командна робота, мульти-бізнес, розширена аналітика

  Оплата: LiqPay (картка) + Крипта (USDT та ін.)
  Перевірка лімітів: plan-guard.js

================================================================
  КОМАНДНА РОБОТА (Workspace)
================================================================

  - Власник бізнесу створює "workspace"
  - Запрошення учасників за унікальним кодом
  - Ролі: Власник / Учасник (Worker)
  - Гнучкий доступ до модулів для кожного учасника
  - Мульти-бізнес: Business-план дозволяє кілька бізнесів
    з ізольованими даними

  МУЛЬТИ-БІЗНЕС

================================================================
  СТРУКТУРА ДАНИХ FIREBASE (Firestore)
================================================================

  users/{uid}                    — профіль користувача
  users/{uid}/{module}/          — дані кожного модуля
    ├── clients/
    ├── invoices/
    ├── projects/
    ├── tasks/
    ├── contracts/
    ├── finances/
    ├── notes/
    ├── passwords/
    ├── appointments/
    ├── services/
    ├── services_cats/           — категорії послуг
    ├── warehouse/
    ├── portfolio/
    ├── hr/
    ├── documents/
    ├── api-keys/
    ├── accounts/
    ├── kanban/
    └── templates/

  workspaces/{workspaceId}       — командний простір
  workspaces/{workspaceId}/members/ — учасники
  tickets/                       — заявки підтримки (глобально)
  announcements/                 — новини WorkHub (глобально)

  UI/UX ОСОБЛИВОСТІ
